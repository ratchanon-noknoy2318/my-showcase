# KPPMCH Digital Service - Promotional Page

This repository contains the source code for a promotional landing page, designed in a classic newspaper style, to showcase the **KPPMCH Digital Service Hub**.

**Live Demo:** [https://kppmch-service.vercel.app/](https://kppmch-service.vercel.app/)

This project was built to serve as a visually engaging introduction to the main service platform developed for the Kamphaeng Phet Municipal Community Hospital.

## About the Main Platform (KPPMCH Digital Service Hub)

The actual service hub is a comprehensive internal platform designed to streamline administrative tasks and improve operational efficiency for hospital personnel. Its core services include:

- General Medical Examination
- Traditional Chinese Medicine (Acupuncture)
- Traditional Thai Medicine
- Telemedicine Service
- Antenatal Care (ANC) Clinic
- And other public health services.

## Technologies Used (This Promotional Page)

- **Framework:** [Next.js](https://nextjs.org/)
- **Library:** [React](https://reactjs.org/)
- **Styling:** CSS Modules, with a design inspired by classic newspapers.
- **Fonts:** [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) & [Noto Serif Thai](https://fonts.google.com/specimen/Noto+Serif+Thai) via Google Fonts.
- **Deployment:** [Vercel](https://vercel.com/) & [GitHub Pages](https://pages.github.com/).

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ratchanon-noknoy2318/resume.git
    cd resume
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
